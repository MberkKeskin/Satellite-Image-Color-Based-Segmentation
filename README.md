# Satellite-Image-Color-Based-Segmentation
HSV-based satellite image segmentation for vegetation, water, cloud, and urban area detection with area ratio analysis

This project performs simple HSV-based segmentation on satellite images.

It classifies regions into:
-  Vegetation
-  Water
-  Cloud
-  Urban

---

##  Features

- HSV color-space segmentation
- Area ratio calculation (%)
- Colored visualization output
- CLI support
- Save output image
- Optional visualization
